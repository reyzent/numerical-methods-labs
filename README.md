## Лабораторная работа №8
### Нахождение решений обыкновенных дифференциальных уравнений численными методами

**Цель работы:** Изучить численные методы приближенного решения обыкновенных дифференциальных уравнений.

**Исходные данные:**
- **Вариант:** 10
- **Уравнение:** `y' = x*exp(-x²) - 2xy`
- **Использованные методы:** 
  - Метод 1: Метод Эйлера
  - Метод 5: Метод Рунге-Кутта четвертого порядка
- **Шаги интегрирования:** `h = 0.01` и `h = 0.001`
- **Начальные условия:** `x₀ = 0, y₀ = 1` (выбраны самостоятельно)
- **Параметры:** Параметры `a` и `b` в уравнении отсутствуют
